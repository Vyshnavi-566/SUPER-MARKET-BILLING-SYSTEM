**Supermarket Billing System**
A simple Java application that helps automate billing in a supermarket.
This project is used to:
    -> View available products
    -> Select items using product ID and quantity
    -> Calculate subtotal, GST, and final bill
    -> Store and retrieve product details from a MySQL database
**Main Functionalities**
Login System - Allows secure access to the application.
Product Display - Shows the list of products stored in the database.
Billing Process - User enters product ID and quantity
System calculates the price and GST
Generates final bill with date and time
Validation : 
If product ID doesn’t exist → shows error
If quantity is more than stock → shows warning
Handles invalid or wrong inputs 
Unit Testing : Subtotal calculation,GST calculation,Stock check
